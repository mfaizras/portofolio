Inizio Woman

Overview

Inizio Woman is a fashion rental platform offering curated clothing for various occasions. The project began as a static catalog distributed via shared files. I rebuilt it from the ground up into a fully dynamic web platform with a complete system, enabling non-technical teams to manage products, categories, images, and details without touching a single line of code.

What I Did

From Static Files to a Dynamic Platform

Transformed a static, manually-shared catalog into a proper web application backed by a database-driven system.

Flexible Product & Variant System

Designed a relational data model capable of handling the complexity unique to clothing rentals — where a single item can have multiple attributes such as size, color, etc.

Rental Availability & Booking Logic

Built the rental availability system that displays only open and available rental schedules to customers in real time. Rather than a self-checkout flow, customers browse available dates on the platform and finalize their booking through the admin team via WhatsApp — keeping the process personal while the system handles availability tracking and conflict prevention on the backend.
